# NetToolkit v1.0

> A lightweight Windows desktop toolkit for network diagnostics and analysis.

---

## Features

### 🔢 Subnet Calculator
Enter an IP address and subnet mask or CIDR notation to instantly compute:
- Network address, broadcast address, CIDR
- Subnet mask, first host, last host
- Number of usable hosts, IP class, range type (Private/Public)
- Full **binary breakdown** of IP, mask, network, and broadcast

### 🔍 Network Scanner
Scan any IP range on your local network and retrieve per-host details:
- Hostname, MAC address, OS (detected via TTL)
- Open ports (SSH, HTTP, RDP, and others)
- Ping response time and online/offline status
- Persistent **scan history** across sessions
- **Export** results for reporting or documentation

### 🌐 IP Info Lookup
Look up geolocation and network info for any public IP address — or leave blank to query your own:
- City, region, country
- ISP and organisation
- Timezone and coordinates
- Proxy/VPN detection and mobile network flag
- One-click **Copy** to clipboard
